#### Panel

- Cleaned up `k-toolbar` component [#82](https://github.com/getkirby/v4/pull/82)
- Drawer Component Structure
```html
<k-overlay type="drawer">
	<form class="k-drawer" method="dialog">
		<k-drawer-notification />
		<k-drawer-header />
		<k-drawer-body>
			<k-drawer-fields />
		</k-drawer-body>
	</form>
</k-overlay>
```

- Portals: Dialogs, Drawers and other Overlays are now seperated into different portals. This is done by setting the overlay type:
	- `<k-overlay type="dialog">`
	- `<k-overlay type="drawer">`
	- `<k-overlay>`
	An overlay without a type will use the default overlay portal.Seperating them into different portals gives us more control over z-index and layering of elements.
