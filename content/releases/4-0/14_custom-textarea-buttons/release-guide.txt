Example:

```js "/site/plugins/acme-textarea"
panel.plugin("acme/textarea", {
  textareaButtons: {
    highlight: {
      label: "Highlight",
      icon: "wand",
      click: () => this.command("toggle", "<mark>", "</mark>")
      shortcut: "m"
    }
  },
});
```
