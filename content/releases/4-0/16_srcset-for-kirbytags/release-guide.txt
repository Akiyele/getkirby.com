## With absolute sizes

```kirbytext
(\image: foo.jpg srcset: 200, 300)
```

## With a config srcset name

https://getkirby.com/docs/reference/system/options/thumbs#srcsets

```kirbytext
(\image: foo.jpg srcset: album)
```
