Title: Default Page Model

----

Subtitle: Set a new page standard

----

Category: core

----

Intro: The DefaultPage class is the standard for all pages that don't have a specific page model.

----

Teaser:

The default page model kicks in, when there is specific model for a page.

Of course, other models can extend the default page model, too.

----

Nolt:

----

Github: https://github.com/getkirby/v4/pull/10

----

Text:

You can define a DefaultPage class in a plugin or in `/site/models/default.php`. This model will be loaded when no other model is defined for the current page type.

```php "/site/models/default.php"
<?php

use Kirby\Cms\Page;

class DefaultPage extends Page
{
	// your custom methods
}
```

Other page models can extend this default model. For this to work reliably, define the `DefaultPage` class in a plugin.
